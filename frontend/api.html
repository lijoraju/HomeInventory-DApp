Smart Contract Functions

createItem

Create a new item in the inventory.

Function Signature

function createItem(
    string memory _name,
    string memory _description,
    uint256 _purchaseDate,
    uint256 _purchasePrice,
    string memory _category,
    string memory _model,
    string memory _serialNumber,
    string memory _location,
    string memory _condition
) external onlyOwner

Parameters

  • _name: Name of the item.
  • _description: Description of the item.
  • _purchaseDate: Purchase date of the item (UNIX timestamp).
  • _purchasePrice: Purchase price of the item (in wei).
  • _category: Category of the item.
  • _model: Model of the item.
  • _serialNumber: Serial number of the item.
  • _location: Location of the item.
  • _condition: Condition of the item.

Outputs

No direct outputs. The item is created and stored in the contract.

Example

// Assuming you have a contract instance and connected web3

const itemName = "Laptop";
const itemDescription = "High-performance laptop";
const purchaseDate = Math.floor(Date.now() / 1000); // Current timestamp
const purchasePrice = web3.utils.toWei("1000", "ether"); // Convert ETH to wei
const category = "Electronics";
const model = "XYZ123";
const serialNumber = "ABC123";
const location = "Home Office";
const condition = "New";

await contractInstance.methods.createItem(
    itemName,
    itemDescription,
    purchaseDate,
    purchasePrice,
    category,
    model,
    serialNumber,
    location,
    condition
).send({ from: accounts[0] });

updateItem

Update details of an existing item.

Function Signature

function updateItem(
    uint256 _id,
    string memory _name,
    string memory _description,
    uint256 _purchaseDate,
    uint256 _purchasePrice,
    string memory _category,
    string memory _model,
    string memory _serialNumber,
    string memory _location,
    string memory _condition
) external onlyOwner

Parameters

  • _id: ID of the item to update.
  • _name: Updated name of the item.
  • _description: Updated description of the item.
  • _purchaseDate: Updated purchase date of the item (UNIX timestamp).
  • _purchasePrice: Updated purchase price of the item (in wei).
  • _category: Updated category of the item.
  • _model: Updated model of the item.
  • _serialNumber: Updated serial number of the item.
  • _location: Updated location of the item.
  • _condition: Updated condition of the item.

Outputs

No direct outputs. The item's details are updated in the contract.

Example

// Assuming you have a contract instance and connected web3

const itemIdToUpdate = 1; // ID of the item to update
const updatedName = "Updated Laptop";
const updatedDescription = "High-performance laptop with new features";
const updatedPurchaseDate = Math.floor(Date.now() / 1000); // Current timestamp
const updatedPurchasePrice = web3.utils.toWei("1200", "ether"); // Convert ETH to wei
const updatedCategory = "Electronics";
const updatedModel = "XYZ456";
const updatedSerialNumber = "ABC123";
const updatedLocation = "Home Office";
const updatedCondition = "Used";

await contractInstance.methods.updateItem(
    itemIdToUpdate,
    updatedName,
    updatedDescription,
    updatedPurchaseDate,
    updatedPurchasePrice,
    updatedCategory,
    updatedModel,
    updatedSerialNumber,
    updatedLocation,
    updatedCondition
).send({ from: accounts[0] });

getMyItems

Retrieve items owned by the caller.

Function Signature

function getMyItems() external view returns (Item[] memory)

Outputs

Returns an array of Item structs representing items owned by the caller.

Example

// Assuming you have a contract instance and connected web3

const ownedItems = await contractInstance.methods.getMyItems().call();
console.log(ownedItems);

Item Structure

  • name: Name of the item.
  • description: Description of the item.
  • purchaseDate: Purchase date of the item (UNIX timestamp).
  • purchasePrice: Purchase price of the item (in wei).
  • category: Category of the item.
  • model: Model of the item.
  • serialNumber: Serial number of the item.
  • location: Location of the item.
  • condition: Condition of the item.
  • owner: Address of the item's current owner.
  • id: ID of the item.

startAuction

Start an auction for an item.

Function Signature

function startAuction(uint256 _itemId, uint256 _basePrice, uint256 _durationInHours) external onlyOwner

Parameters

  • _itemId: ID of the item to start the auction for.
  • _basePrice: Base price of the auction (in wei).
  • _durationInHours: Duration of the auction in hours.

Outputs

No direct outputs. An auction is started for the specified item.

Example

// Assuming you have a contract instance and connected web3

const itemIdToStartAuctionFor = 1; // ID of the item to start the auction for
const basePrice = web3.utils.toWei("100", "ether"); // Convert ETH to wei
const auctionDurationInHours = 24; // Duration of the auction in hours

await contractInstance.methods.startAuction(itemIdToStartAuctionFor, basePrice, auctionDurationInHours).send({ from: accounts[0] });

placeBid

Place a bid in an ongoing auction.

Function Signature

function placeBid(uint256 _itemId, uint256 _bidAmount) external payable

Parameters

  • _itemId: ID of the item in the auction.
  • _bidAmount: Bid amount in wei.

Outputs

No direct outputs. A bid is placed in the auction for the specified item.

Example

// Assuming you have a contract instance and connected web3

const itemIdToPlaceBidFor = 1; // ID of the item to place a bid for
const bidAmount = web3.utils.toWei("150", "ether"); // Convert ETH to wei

await contractInstance.methods.placeBid(itemIdToPlaceBidFor, bidAmount).send({ from: accounts[1], value: bidAmount });
