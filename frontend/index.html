About HomeInventory DApp

HomeInventory DApp is a decentralized application built on the blockchain technology and web3.0 that allows you to manage your home inventory and participate in auctions for various items. You can start auctions, place bids, transfer ownerships, and more.

How to Use the DApp

To use the HomeInventory DApp, simply navigate to the respective pages using the navigation links above. Follow the instructions on each page to interact with the HomeInventory smart contract on the blockchain.

Requirements

To use this DApp, you need to have the MetaMask extension installed in your web browser. Make sure you have an Ethereum account and some Ether for interacting with the blockchain.
